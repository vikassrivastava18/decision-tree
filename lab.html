How to build a Decision Tree

We build a decisions tree recursively.

  1. Calculate Entropy For a selected feature
  2. Calculate Information Gain or Gini Impurity
  3. Select the feature which performs best on above criteria
  4. Repeat the process for next node untill you cross a threshold or reach the final goal.

Pseudo Code

if all examples are in the same class then
    return the class label.
else if no features left then
    return the majority decision.
else if no examples left then
    return the majority decision at the parent node.
else
    choose a feature f.
for each value v of feature f do
    build edge with label v.
    build sub-tree using examples where the value of f is v.
